# Next.Js-Guide

A complete, hands-on reference project covering essential Next.js features with TypeScript and Tailwind CSS. It is structured as a beginner-friendly learning guide for developers with little or no React or Next.js experience.

## What this project includes

- App Router pages and layouts
- Server and Client Components
- Data fetching patterns
- API routes
- Forms with Server Actions and client-side validation
- Images, styling, and TypeScript examples
- GitHub and deployment guidance
- Light and dark theme support

## Getting started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Production build

To verify the app builds correctly for production:

```bash
npm run build
npm run start
```

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Purpose

This repo is intended to be both:

- a learning document for understanding core Next.js concepts
- a reference starter for future projects
